# Single-Page-CV

A single-page CV project based on the challenge from [roadmap.sh](https://roadmap.sh/projects/single-page-cv).

## How to Run

1. **Clone or download** this repository to your local machine.
2. Open the project folder.
3. Locate the `index.html` file.
4. Open `index.html` in your browser:
   - Double-click the file, **or**
   - Right-click and select **"Open with" â†’ your preferred browser**, **or**
   - Run a local web server (optional, for testing JavaScript/CSS properly):
     ```bash
     # Using Python 3
     python3 -m http.server
     ```
     Then open your browser and go to [http://localhost:8000](http://localhost:8000).
